    Two tags
  1. The title is one of several elements that provide extra information not visible on the web page, but it is useful for search engines or how the page gets displayed.
  2. You can add style to an element by specifying it in the style element within head tag
  3. The div element is used mainly for design layout purposes unlike the other content elements
  4. pecentage elements in CSS will be deducted from the parent element; example 80% width of div thats inside another div will take 80% of that div width
  5. if you want to center any element horizontally. You can do this by setting its margin-left and margin-right properties to auto.
  6. article elements commonly contain multiple elements that have related information
  7. width & max-width elements are used together to prevent width of element of exceeding too wide.
  8. You change properties of a link when the link has actually been visited by using a pseudo-selector that looks like a:visited { propertyName: propertyValue; }.
